### 🗑 Background?

```

쓰레기는 올바른 방식으로 처리되어야 하는 것이 맞으나 해당 지역에서 판매하는 종량제봉투에 담아서만 처리가 가능하다. 
하루의 쓰레기 처리를 위해 종량제 봉투를 따로 구입할 수도 없고, 
그렇다고 공공 쓰레기통을 찾을 때까지 쓰레기를 들고 돌아다니기에도 애매한 문제이다.

```
